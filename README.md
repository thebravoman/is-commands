# Is::Commands
Commands framework for Axles IS. See {@link IS.Commands}

## Usage
```html
<script src=".../ext/commands/register_commands"> </script>
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'is-commands'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install is-commands
```
